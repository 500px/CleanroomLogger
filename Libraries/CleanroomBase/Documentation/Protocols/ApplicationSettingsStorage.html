ApplicationSettingsStorage

This protocol extends ApplicationSettingsProvider by adding the ability to store and remove values. The DefaultApplicationSettingsStorage class provides the default implementation of this protocol. Consider using an instance of the class ApplicationSettingsWriter for writing values to an ApplicationSettingsStorage instance.
